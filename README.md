# Food Store — Sistema de Gestión de Pedidos de Comida

## Descripción
Sistema de consola desarrollado en Java 21 que permite gestionar categorías, productos, usuarios y pedidos mediante un menú interactivo.

## Tecnologías utilizadas
- Java 21
- NetBeans IDE

## Cómo ejecutar el proyecto
1. Clonar o descargar el repositorio
2. Abrir el proyecto en NetBeans
3. Ejecutar la clase Main.java

## Estructura del proyecto
- `foodstore.entities` → Clases del modelo (Base, Categoria, Producto, Usuario, Pedido, DetallePedido)
- `foodstore.enums` → Enumerados (Rol, Estado, FormaPago)
- `foodstore.exception` → Excepciones personalizadas
- `foodstore.service` → Lógica de negocio
- `foodstore.ui` → Menú de consola y clase Main
